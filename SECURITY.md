# 🔒 Security — VantaPoint

VantaPoint is designed under a no-compromise security model for verified, high-clearance users.

## 🔐 Core Protocols
- **Tiered Role Enforcement** — Each action tied to access level + verification
- **Encrypted Death Maps** — Biometric + secondary token required
- **Lockdown Browser** — No other apps/functions accessible during operation
- **Zero Shared Backends** — Fully separate infra from Odyssi

## 🧱 Encryption Standards
- AES-256 by default
- Biometric encryption layer
- Optional FIPS 140-2 hardware binding
- Obfuscated data paths and filenames

## 💣 Anti-Breach Features
- Geo/time-expiry missions
- Self-destruct triggers on tamper
- Per-device hardware fingerprinting
- Admin ghost logins with forensic logs

## 🧪 Known Limitations
- App will not function on rooted or jailbroken devices
- Currently no fallback if hardware is destroyed unless external storage module used
